  • A certified copy of the ID card of the securities holder requesting the change of mailing address (the ID card must not expire)
  • If the name and surname have been changed: enclose a certified copy of the name change document
  • An original ID card or certificate issued by the district office (no older than 90 days) that can be used in lieu of the original ID if the application documents are sent by post or through an authorized person.
  • The securities certificate with the previous name
  • Complete the application form for the issuance of new securities certificates (TSD -102) and enclose one additional set of supporting documents as specified in the form.
